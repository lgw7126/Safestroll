> A senior-controlled outdoor walk check-in app — not family surveillance.

## 앱 개요 · Overview

SafeStroll은 어르신이 외출 시 **귀가 예정 시간을 스스로 설정**하고,
귀가 후 직접 체크인하는 자기주도형 안전 앱입니다.

가족이 위치를 추적하는 것이 아니라, **어르신이 원할 때만 연락처에 알림이 전송**됩니다.

---

## 주요 기능 · Features

| 기능 | 설명 |
|---|---|
| **외출하기 버튼** | 큰 버튼 한 번으로 외출 시작 |
| **귀가 시간 설정** | 30분 / 1시간 / 1.5시간 / 2시간 빠른 선택 또는 직접 입력 |
| **실시간 카운트다운** | 귀가까지 남은 시간 `HH:MM:SS` 표시 |
| **안전 귀가 체크인** | "집에 돌아왔어요!" 버튼으로 완료 |
| **자동 알림** | 귀가 시간 초과 시 비상연락처에 브라우저 알림 전송 |
| **외출 기록** | 날짜, 출발·귀가 시간, 안전 귀가 여부 기록 |
| **비상연락처 설정** | 이름 + 전화번호 저장 (직접 전화 연결 링크 포함) |

## 화면 구성 · Screens

```
홈 화면            → 외출하기 버튼
  ↓
시간 설정 화면     → 귀가 예정 시간 선택 → 출발!
  ↓
외출 중 화면       → 카운트다운 + 집에 돌아왔어요! 버튼
  ↓ (시간 초과 시)
알림 화면          → 비상연락처에 알림 발송 + 귀가 체크인 가능
  ↓
홈 화면            → 외출 기록에 자동 저장
```

## 기술 스택 · Tech Stack

- **React 18** — UI 컴포넌트
- **Vite** — 빌드 도구
- **localStorage** — 브라우저 내 데이터 저장 (서버 불필요)
- **Web Notifications API** — 브라우저 알림
- **Noto Sans KR** — 한글 최적화 폰트
- **CSS 변수** — 테두리(teal) + 크림(cream) 2색 팔레트

> SMS 문자 발송은 Twilio 등 서버 연동 시 추가 가능합니다.

---

## 디자인 원칙 · Design Principles

- **최소 버튼 높이 80px** — 어르신 터치 편의성
- **2가지 색상** — 진한 테두리(#2E7D6B) + 크림(#F5F0E8), 차분하고 알림 없는 디자인
- **한국어 + 영어 병기** — 모든 버튼에 한국어(크게) + 영어(작게)
- **오프라인 지원** — 인터넷 없이도 동작, 새로고침 후 외출 상태 복원
- **어르신 자율성** — 위치 추적 없음, 어르신이 직접 시간 설정·체크인
